Contoh Text Alignment CSS

Text Left

Paragraf ini menggunakan text-align left. Teks akan rata ke sebelah kiri dan biasa digunakan untuk penulisan artikel atau paragraf.

Text Center

Paragraf ini menggunakan text-align center. Biasanya dipakai untuk judul, slogan, atau teks penting agar terlihat menonjol.

Text Right

Paragraf ini menggunakan text-align right. Umumnya digunakan untuk tanda tangan atau teks tertentu.

Text Justify

Paragraf ini menggunakan text-align justify. Teks akan rata kiri dan kanan sehingga terlihat lebih rapi seperti di buku atau koran.
