teks formating untuk teks dari komputerL

Printah javascript untuk menampilkan teks ke console adalah console.log(). Kita juga biasa menampilkan isi variabel dengan fungsi ini. Misalkan kita piunya variabel name, maka kode program bisa ditulis seperti ini: 

        var name = "kelas X PPLG 2";
        console.log(name)
maka hasil outputnya: kelas x PPLG 2

untuk menjalankan ulang program, lakukan refresh dengan menekan tombol F5
